Добро пожаловать в инвестиционного помощника! Бот имеет следующий функционал:
1.Узнать текущую стоимость актива
2.Получить подробный анализ актива
3.Добавление/удаление актива из портфеля. 
4.Выставление таймера на обновление информации по активам из портфеля
Уведомления по активам из портфеля приходит в виде краткого сообщения с текущими показателями
Для взаимодействия с ботом вы можете использовать клавиатуру, на которой всегда будут подсказки по дальнейшим шагам, или команды «/help», «/menu».
Для работы с ботом, перейдите в меню
